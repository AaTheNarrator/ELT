Задание на ELT процесс: Импорт данных о фильмах и их актерах

Цель задания: Создать базу данных для хранения информации о фильмах и актерах, а затем загрузить данные из источника в эту базу данных с помощью ELT процесса.

Шаги задания:

1. Создание базы данных:

Создайте базу данных с названием "MoviesDB" (или любым другим названием на ваш выбор).
2. Создание таблиц:

Создайте две таблицы: "Movies" и "Actors". Вот структура таблиц:
Таблица "Movies":

movie_id (Уникальный идентификатор фильма, целое число, автоинкремент)
title (Название фильма, строка)
year (Год выпуска фильма, целое число)
director (Режиссер фильма, строка)
Таблица "Actors":

actor_id (Уникальный идентификатор актера, целое число, автоинкремент)
name (Имя актера, строка)
birthdate (Дата рождения актера, дата)
3. Импорт данных:

Загрузите тестовые данные в формате CSV (или любом другом удобном формате) с информацией о фильмах и актерах. Пример данных:
Данные для таблицы "Movies":

sql
Copy code
movie_id, title, year, director
1, "The Shawshank Redemption", 1994, "Frank Darabont"
2, "The Godfather", 1972, "Francis Ford Coppola"
3, "Pulp Fiction", 1994, "Quentin Tarantino"
Данные для таблицы "Actors":

arduino
Copy code
actor_id, name, birthdate
1, "Morgan Freeman", 1937-06-01
2, "Marlon Brando", 1924-04-03
3, "John Travolta", 1954-02-18
4. ELT процесс:

Извлеките данные из файлов CSV.
Преобразуйте данные (например, преобразуйте даты в нужный формат).
Загрузите данные в соответствующие таблицы базы данных "MoviesDB".
5. Завершение задания:

Проверьте, что данные успешно загружены в базу данных "MoviesDB".
Проведите несколько SQL-запросов для проверки корректности данных, например, выведите список всех фильмов или актеров.
Это задание поможет вам настроить процесс ELT и понять, как данные из источника могут быть преобразованы и загружены в базу данных для дальнейшего анализа.
